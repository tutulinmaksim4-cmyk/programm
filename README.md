# Проект по Банковским операциям
Этот проект преднозначен для сортировки и фильтрации банковских данных
## Как работает проект
- модуль "masks" маскирует номер банковской карты
- так же "masks" принимает на вход номер счета и возвращает его маску формате
    **XXXX

- модуль "processing" фильтрует данные по значениям (EXECUTED/CANCELED)
- так же "processing" сортирует данные по дате
